===Slot Machine Mod for MineTest===
by Alex Carroll and Andres Lucca

This mod adds a slot machine block and the recipe to create it. 

Recipe: 

	S  S  S
	S  G  S
	S  S  S

	S = Steel Ingot  G = Gold Ingot
